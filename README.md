# Aditya Raj Singh Chouhan - Portfolio

A futuristic, animated personal portfolio website built with React, Vite, and Tailwind CSS.

## 🚀 Features

- **Stunning Animations** - Smooth Framer Motion animations throughout
- **Particle Background** - Interactive particle system using tsparticles
- **Custom Cursor** - Unique cursor effects on desktop
- **Glassmorphism Design** - Modern glass-effect UI components
- **Fully Responsive** - Works perfectly on all devices
- **Dark Theme** - Eye-friendly dark mode design
- **Smooth Scrolling** - Seamless navigation experience

## 🛠️ Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Framer Motion
- React Icons
- tsparticles

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌐 Deployment

This portfolio can be deployed to:
- **Vercel** - Just connect your GitHub repo
- **Netlify** - Drag and drop the `dist` folder
- **GitHub Pages** - Use the `gh-pages` branch
- **Any static hosting** - Upload the `dist` folder

### Build Command
```bash
npm run build
```

### Output Directory
```
dist/
```
